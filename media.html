Les media Intégrés

Les images

img

img est une balise qui une fois écrite, va aller chercher l'image source et l'afficher à sa place. Si rien d'autre n'est précisé, l'image s'affiche à sa taille originale. Si on change, par exemple, sa largeur, la hauteur garde le ratio (l'homothétie, le rapport hauteur / largeur)

Une image est en display inline, c'est à dire que si on l'intègre à l'intérieur d'un paragraphe : , le texte se placera à gauche et à droite de l'image : elle est intégrée au "flux" du texte.

Il est possible d'imposer une largeur et une hauteur à une image, mais il faudra utiliser une classe css dans laquelle on précise que l'objet (ici le flux de l'image) "couvre" l'espace proposé.

Figure

Il existe une balise conteneur pour les media, qui aide la sémantique web pour le référencement et l'accessibilité. la balise figure va encadrer un media, et donne la possibilité d'y ajouter une légende, avec la balise figcaption.

Picture

La balise picture permet de faire de l'affichage d'image adaptatif : selon la taille disponible autour de l'image affichée, on peut sélectionner la même image dans des formats différents.

Audio et vidéo

audio

Pour mettre à disposition un fichier audio sur un site, on utlise la balise audio. Cette balise peut soit appeler un seul fichier, ou utiliser aussi la balise source. Utiliser plusieurs sources permet de s'adapter à la configuration de l'utilisateur. Un appareil, pour lire un fichier audio (ou vidéo) doit posséder les bon codecs. Si je met a disposition un fichier avec une extension que l'appareil ne connaît pas, j'aurais une erreur. Si je propose le même fichier dans des formats différents avec source, l'appareil lancera le premier fichier qu'il est capable de lire.

video

La balise video se comporte comme la balise audio avec quelques attributs supplémentaires

Exercice : recréer la figure vidéo avec le lecteur poour les deux vidéos nommées "Big_Buck_Bunny_360_10s_1MB.mp4" et "Big_Buck_Bunny_360_10s_1MB.webm". Utilisez l'id "bunny" pour l'élément vidéo.

Les media embarqués

Les cadres de pages

"embarquer" un média signifie que le media n'est pas hébergé sur le site que l'on visite mais sur un site ou hébergement spécialisé pour ce type de media.

iframe

L'élément iframe permet d'inserrer une page internet dans le document, mais pas que, il est possible d'inserrer d'autres types de fichiers, par exemple des pdf ou directement des fichiers vidéos, sons, etc.

L'iframe avait un ancètre, la combinaison frameset et frame.

Images embarquées

Pour les images, certains fora ou réseaux sociaux ne proposent pas directement le téléversement de fichiers dans les publications des utilisateurs, alors on se tourne vers un hébergeur multimédia sur lequel on téléverse les media que l'on "embarque" ensuite sur les publication précédemment citées.

Vidéos embarquées

La plupart des site d'hébergement de vidéos propose un partage "intégré" en utilisant les iframes

Youtube propose un partage avec une hauteur et une lageur imposée, non compatible avec les affichage smartphone en portrait

Dailymotion propose un partage avec un ensemble de balise et de css intégré qui permet l'affichage en adaptatif 16/9

L'audio embarqué

Les site de partage d'audio proposent déjà des iframes faites pour l'adaptatif
